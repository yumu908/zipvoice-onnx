# ZipVoice ONNX

ZipVoice ONNX is a library for performing text-to-speech synthesis using the ZipVoice model.

See ZipVoice on [Hugging Face](https://huggingface.co/k2-fsa/ZipVoice) and [GitHub](https://github.com/k2-fsa/ZipVoice).

## Installation

```console
uv pip install zipvoice-onnx
```

## Usage

```console
uv run examples/english.py
```
